# EnumX-Offensive-Enumeration-Tools
EnumX is a curated collection of offensive security enumeration tools designed for reconnaissance and attack surface mapping during authorized penetration testing engagements. Each tool focuses on extracting critical information from specific platforms and services to support the initial phases of security assessments.
Currently includes enumeration tools for WordPress, Drupal and Joomla, with upcoming additions such as SMB enumeration, exposed-service scanning, and additional offensive utilities.
